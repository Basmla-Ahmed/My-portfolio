# My Portfolio

A personal portfolio website to showcase my projects, skills, and experience. Built with a focus on clean design and responsiveness.

## 🌟 Features

- Responsive layout (works on desktop, tablet, and mobile)
- Project showcase with links to live demos and repositories
- About section with bio and skills
- Contact form (optional)
- Smooth scrolling and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox/Grid, animations)
- JavaScript (ES6)
- Bootstrap
